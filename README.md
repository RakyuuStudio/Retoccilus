## Retoccilus - A Full Solution for Game Development

Hello, and welcome! Welcome to the Repository of Retoccilus!  
Retoccilus is a full kit of game development, includes Core Engine, DAW, 3D-Modeler,
Animator and so many extensions that useful to develop your game and movie(probably)!  

Wish you have fun in Retoccilus!

### How to build

Because Retoccilus depend on [Conan](https://conan.io) and [Qt](https://www.qt.io), you should install them first.  

Then, open Command Prompt / Powershell (Windows) / Shell (Linux / macOS), run:

```Shell
conan install .
```

to download and install the dependencies.  

After install the dependencies, don't forget to change the Qt Prefix Path in the CMakeLists.txt.
