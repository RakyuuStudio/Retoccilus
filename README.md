## Retoccilus - A Full Solution for Game Development

Hello, and welcome! Welcome to the Repository of Retoccilus!  
Retoccilus is a full kit of game development, includes Core Engine, DAW, 3D-Modeler,
Animator and so many extensions that are useful to develop your game and movie(probably)!  

Wish you have fun in Retoccilus!  

### Information

Retoccilus is an engine that uses lots of technologies combined.  
And it's under BSD 3-Clause License. You can be free to copy and distribute. 

### How to build

If you are a normal user that you want to use Retoccilus as the Engine, it's unnecessary to build it from source.  

But, if you are a developer that you want to build Retoccilus from source, you can follow the following steps:

1. Install the dependencies and toolkit listed below:
    - (Qt)[https://www.qt.io/download]
    - [It should be installed belong Qt.] (CMake)[https://cmake.org/download/]
    - (Node.js)[https://nodejs.org/en/download/]
    - (npm)[https://www.npmjs.com/]
