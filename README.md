## Welcome to the Repository of Retoccilus!

Welcome to the Repository of Retoccilus! Retoccilus is a game engine based on Qt6.

This repository contains the source code of Retoccilus, including the Code Editor, Main Program, Physical System, and Extensions.

(Except the Code Editor, and others are developing)

## Features


1. Code Editor: Retoccilus provides you with a functional IDE inside the Game Engine. Contains Auto Parentheses, Syntax Highlighting, Code Auto-completion,

   Code Formatter and lots of helpful features to enhance your coding experience.

2. Engine: The core functionality of Retoccilus. It can handle various tasks such as Game Testing, Managing Assets, Animation Controlling and Rendering.
3. Physical System: An extension includes in Retoccilus core. This physical system includes a robust physical system that simulates the real-world physics. Allows you to apply real-world physics on game objects.
4. External Applications: Retoccilus provide you three useful External Applications, such as 3D Modeling, Music Producing, and Animating.
5. Extensions: Retoccilus gives you a whole functionality Programming Language called RetoExt, using simple functions to control the Retoccilus Core, and make extensions you like.
## Getting Started

To get started with Retoccilus, please follow these steps:

1. Clone the whole repository to your local machine:

   ```bash
   git clone https://github.com/RakyuuStudio/Retoccilus.git
   ```

2. Install these necessary dependencies and SDKs:
- FBXSDK (by Autodesk)
- VST3SDK (by Steinberg Media Technologies)
  Make sure you have installed the whole Qt6 on your computer.
3. Import the project into your favorite build system (Such as Qt Creator and CLion)
4. Build and Run the Software
