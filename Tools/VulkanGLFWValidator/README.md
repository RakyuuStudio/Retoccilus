# `Tools/VulkanGLFWValidator` Folder

This folder contains the source that you can try to run.  

If you can successfully run each source (incudes `GLFWValidator.cpp`, `VulkanValidator.cpp`), that means
your Vulkan Development is successfully being setup.  

You can just create another project, copy the `CMakeLists.txt`, and delete the Qt6 Library Linking, copy the each
source in this folder, if you can see each window without any errors, congrats!
