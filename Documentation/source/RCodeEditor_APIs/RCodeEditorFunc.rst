RCodeEditor Functions
===============================
Welcome to the function reference page of RCodeEditor.

RCodeEditor provide you a fully operational code editor for any languages. Only needs to change configuration file.

The following functions are available in RCodeEditor:

1. `RCodeEditor::RCodeEditor` - Constructor. Create a new instance of RCodeEditor.
2. `initDocumentLayoutHandlers` - Initialize document layout handlers.
3. `initFont` - Initialize font in code editor.
4. `setHighlighter` - Set highlighter for code editor.
5. `setSyntaxStyle` - Set syntax style for code editor.
6. `updateStyle` - Update style for code editor.
7. `onSelectionChanged` - Handle selection change event.
8. `updateLineGeometry` - Update line geometry for code editor.
9. `getFirstVisibleBlock` - Get first visible block in code editor.
10. `setAutoParentheses` - Set auto parentheses for code editor.
