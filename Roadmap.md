## Roadmap of Retoccilus

This is a roadmap of Retoccilus will be developed in future.  

### Version 1.0
#### Milestone 0.0.1 - Basic 3D Scene Support.

- Key Function
    - Support common 3D object format (`.fbx`, `.obj` with `.mtl`, `.glb` (Binary format of GLTF))
    - Support import 3D object into scene.
    - Implement the basic rendering-pipeline with Vulkan.
    - Implement the basic movement of camera(Rotate, Zoom, Move)
- Test
    - Using simple objects to test the scene and object import.GL
    - Test the different angles of camera.
- Documentation
    - Add documentation of import 3D object via Sphinx.
    - Add documentation of control the camera.

#### Milestone 0.0.2 - Basic Shader and Light.

- Key Function
    - Support script to control the camera.
    - Support the basic shader like Phong and Lambert.
    - Support light.
