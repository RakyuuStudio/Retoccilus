# `doc` folder:

This folder contains the document of Retoccilus Platform. Includes the Developer Documentation and User Documentation.  

If you want to host the document server locally, simply run `gen-doc.bat` (On Windows) and `gen-doc.sh` (On macOS and Linux). And the script will open the page directly in your browser.
